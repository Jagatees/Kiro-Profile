# Version 0.2.0 - Complete Stats Implementation

## What's New

This version implements **comprehensive activity tracking** for Kiro IDE with all available stats across 5 new sections.

## New Statistics Tracked

### 📊 Session Statistics

- **Total Turns** - Count of all conversation turns
- **Success Rate** - Percentage of successful turns vs failed
- **Failed Turns** - Number of turns that encountered errors
- **Avg Session Duration** - Average time spent per session
- **Sub-agent Calls** - Number of times sub-agents were invoked

### 🎯 Token Breakdown

- **Total Tokens** - Complete token usage across all sessions
- **Prompt Tokens** - Tokens used in user prompts
- **Generated Tokens** - Tokens generated by the AI
- **Prompt/Generated Ratio** - Efficiency metric
- **Avg Context Window** - Average context window size used

### ⏰ Activity Patterns

- **Most Active Hour** - Your peak productivity time
- **Total Active Days** - Days with any activity
- **Current Streak** - Consecutive days of activity
- **Longest Streak** - Your best streak ever
- **Avg Activity/Day** - Average turns per active day

### 🔧 Top Tools Used

- Tracks the top 5 most frequently used tools/functions
- Displays usage counts for each tool
- Helps understand your workflow patterns

### ⚙️ System Stats

- **Errors Logged** - Count of errors encountered
- **Warnings Logged** - Count of warnings
- **Hooks Installed** - Number of agent hooks configured
- **Powers Installed** - Number of Kiro powers active
- **Extensions Installed** - Number of Kiro extensions

## Technical Implementation

### Data Collection

Extracts comprehensive metrics from:

- **Session files** (`~/.kiro/sessions/*.json`)
- **Workspace sessions** (`%APPDATA%/Kiro/User/globalStorage/`)
- **Token generation logs** (`dev_data/tokens_generated.jsonl`)
- **Session JSON structure** (turn_duration, model_info, tool_name, etc.)

### New Tracking

- Hour-by-hour activity patterns
- Tool usage frequency mapping
- Success/failure rate calculations
- Prompt vs generated token ratios
- Context window usage tracking
- Sub-agent invocation counts

## Data Sources

The extension now reads from:

1. Session JSON files (input/output tokens, turn data)
2. Tool usage from session logs
3. Timestamp data for hourly patterns
4. Error/status fields for success tracking
5. Session start/end times for duration
6. Model information for attribution

## Installation

Install the new version:

```powershell
# Remove old version first if needed
code --uninstall-extension jagatees.kiro-stat

# Install new version
code --install-extension kiro-stat-0.2.0.vsix
```

Or just install directly in Kiro IDE:

1. Extensions view (Ctrl+Shift+X)
2. ... menu → Install from VSIX
3. Select `kiro-stat-0.2.0.vsix`
